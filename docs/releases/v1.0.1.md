# PortPilot v1.0.1

Patch release focused on startup command stability.

## 中文

### 修复
- 修复：删除演示命令 `admin-api` 后，重启应用会被自动加回的问题。
- 现在行为：`admin-api` 仅作为演示条目存在于配置中时才显示；用户删除后不会再被自动回灌。

### 发布产物
- `PortPilot-1.0.1.dmg`
- `PortPilot-1.0.1.zip`

---

## English

### Fixes
- Fixed an issue where the demo command `admin-api` was auto-added again after app restart even if users deleted it.
- New behavior: `admin-api` appears only when it exists in config; once removed by users, it stays removed.

### Artifacts
- `PortPilot-1.0.1.dmg`
- `PortPilot-1.0.1.zip`
