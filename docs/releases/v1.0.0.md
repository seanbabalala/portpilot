# PortPilot v1.0.0

**Pilot your ports**

## 中文（可直接粘贴到 GitHub Release）

### 🎉 首个正式版本
PortPilot 是一个面向 macOS 15+ 的原生菜单栏端口管家：实时看端口、看进程、快速启动/停止常用命令，减少终端来回切换。

### ✨ 本次发布亮点
- 菜单栏实时摘要：`: N`，支持异常阈值后显示 `: —`
- 基于 `lsof` 的监听端口扫描：`lsof -nP -iTCP -sTCP:LISTEN`
- 端口列表与搜索：Port / Process / PID / User，稳定排序与新条目高亮
- 小白友好可读化：用途说明、别名、父进程链、启动来源（launchd/brew/docker/ssh）
- 右键快捷操作：复制 URL、复制 PID、复制 kill 命令
- 启动命令管理（核心特色）：将运行中进程一键保存为 YAML 启动项，并支持启动/停止/重启/改端口
- Workspace 场景化管理：通过标签分组命令并一键控制
- 诊断导出：导出监听快照、配置与运行事件，便于排障与分享
- 中英双语切换与资源徽标（CPU/内存）显示

### 🔒 安全默认策略
- 默认不展示完整命令行参数
- 默认禁用 Kill
- Kill 需显式开启并二次确认

### 📦 安装与分发
- 本地打包：`./scripts/package_release.sh`
- 产物：`dist/PortPilot-<version>.dmg`、`dist/PortPilot-<version>.zip`
- 如需通过 Gatekeeper 免拦截分发，请使用 Developer ID 签名 + notarization（仓库已提供脚本）

### ✅ 兼容性
- macOS 15+
- Xcode 16+（源码构建）

### 🙏 致谢
感谢所有测试与反馈，欢迎继续提交 Issue / PR 共建 PortPilot。

## English (ready for GitHub Release)

### 🎉 First stable release
PortPilot is a native macOS 15+ menu bar port companion: monitor listening ports in real time, understand processes faster, and run common startup commands without terminal friction.

### ✨ Highlights
- Real-time menu bar summary: `: N`, with failure-threshold fallback `: —`
- Listener scan powered by `lsof`: `lsof -nP -iTCP -sTCP:LISTEN`
- Searchable port list: Port / Process / PID / User, with stable sorting and “New” highlighting
- Human-readable process context: usage hints, aliases, parent chain, and launch source (launchd/brew/docker/ssh)
- Context actions: Copy URL, Copy PID, Copy kill command
- Startup command management (core differentiator): save live processes into YAML profiles, then start/stop/restart/rebind ports
- Workspace scene control: group commands via tags and operate them in one click
- Diagnostic export: listeners snapshot, config, and runtime events for troubleshooting
- Global CN/EN language switch and CPU/memory resource badges

### 🔒 Secure defaults
- Command-line details are hidden by default
- Kill action is disabled by default
- Kill requires explicit enablement and confirmation

### 📦 Installation & distribution
- Local package build: `./scripts/package_release.sh`
- Outputs: `dist/PortPilot-<version>.dmg`, `dist/PortPilot-<version>.zip`
- For smooth public distribution through Gatekeeper, use Developer ID signing + notarization (scripts included)

### ✅ Compatibility
- macOS 15+
- Xcode 16+ (for source builds)

Thanks to everyone testing and sharing feedback. Issues and PRs are welcome.
