# PortPilot Roadmap（Weekly Ship Plan）

> Last updated: 2026-03-06  
> Start date: 2026-03-09 (Monday)

本路线图用于对外展示每周更新节奏。  
完成项直接从 `[ ]` 改成 `[x]`，并追加 PR / Release 链接。

## 目标（对用户价值）
- 从“看端口”升级为“开发工作台一键启动与自愈”
- 让小白也能理解进程用途并安全管理
- 建立可复用的团队级本地开发场景（Workspace）

## 对外承诺（每周）
- 每周 1 次可下载版本更新
- 每周至少 1 个可感知新能力 + 1 个体验修复
- 每周发布中英双语 Release Notes

## 12 周重磅计划（从 2026-03-09 开始）

- [ ] **W1 · 2026-03-09 ~ 2026-03-15**  
  **主题：竞争基线 + 指标系统**  
  交付：内置产品指标（启动成功率、冲突修复时长、平均恢复时间）、公开 Roadmap 看板、性能基线报告

- [ ] **W2 · 2026-03-16 ~ 2026-03-22**  
  **主题：Workspace DAG 启动编排**  
  交付：服务依赖顺序启动、健康检查闸门、失败自动停止后续依赖

- [ ] **W3 · 2026-03-23 ~ 2026-03-29**  
  **主题：端口改写 2.0（模板化）**  
  交付：Next/Vite/Nest/Uvicorn 常见参数改端口模板、失败回滚

- [ ] **W4 · 2026-03-30 ~ 2026-04-05（重磅）**  
  **主题：一键恢复开发台（Workspace Scene）**  
  交付：一键启动“前端+后端+数据库+隧道”、保存/恢复上次工作状态

- [ ] **W5 · 2026-04-06 ~ 2026-04-12**  
  **主题：进程人话化 2.0**  
  交付：用途标签增强、风险提示、常见解释卡片（面向非 Geek 用户）

- [ ] **W6 · 2026-04-13 ~ 2026-04-19**  
  **主题：Port Timeline**  
  交付：端口占用时间线、谁先占用、冲突回放与诊断导出增强

- [ ] **W7 · 2026-04-20 ~ 2026-04-26**  
  **主题：Docker / Brew 深度联动**  
  交付：容器端口映射展示、brew services 状态控制、来源识别提升

- [ ] **W8 · 2026-04-27 ~ 2026-05-03（重磅）**  
  **主题：冲突自动修复引擎**  
  交付：自动推荐空闲端口、自动 patch 启动命令、自动健康验证

- [ ] **W9 · 2026-05-04 ~ 2026-05-10**  
  **主题：团队共享 Workspace**  
  交付：配置导入导出（脱敏）、模板版本化、团队标准场景同步

- [ ] **W10 · 2026-05-11 ~ 2026-05-17**

  交付：离线“这是什么进程/能否安全结束”解释引擎（默认本地，不上传命令行隐私）

- [ ] **W11 · 2026-05-18 ~ 2026-05-24**  
  **主题：CLI / CI 守卫**  
  交付：启动前端口检查 CLI、CI 端口冲突预警、团队规范化检查

- [ ] **W12 · 2026-05-25 ~ 2026-05-31（重磅发布）**  
  **主题：PortPilot Studio 1.1**  
  交付：场景编排 + 自动修复 + 时间线完整闭环，发布 1.1 说明页

## 更新规则（如何打勾）

- 若延期：在该周末尾追加 `延期到：YYYY-MM-DD`
